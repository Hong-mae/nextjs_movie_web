## 프로젝트 소개

- Next.js, TDMB Api를 이용한 토이 프로젝트입니다.
- 전체적인 스타일은 Netflix를 참고했습니다.
- TMDB Api를 통해 상영 중, 인기 영화, 최고 평점, 최근 개봉 및 임박한 영화를 확인 할 수 있습니다.

## 시작 가이드

### 요구사항

빌드시 필요한 요구사항

- Node.js > 18.18.x
- Npm > 10.x.x
- Next.js > 15.1.6
- Material UI > 6.4.1
- React Player > 2.16.0
- React Slidk > 0.30.3
- Slick Carousel > 1.8.1
- Zustand > 5.0.3
